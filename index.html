What is Neovim?

Neovim is an extremely customisable text editor known for its lightning-fast loading speed. It's an upgrade from the classic Vim editor, offering a more modern interface and additional features.

It allows its users to use various plugins to fully tailor their experience to their needs by using a programming language called Lua, unlike Vim, which uses its own scripting language, Vimscript.

Also, Neovim increases developers' productivity by allowing them to rely fully on their keyboard. You do NOT need a mouse to use Neovim. Every command and motion can be executed using the keyboard.

How to navigate around Neovim?

Neovim has three modes that can be used:

  • Normal mode: Used for moving around Neovim, deleting, yanking(copying) and pasting text (activated by pressing the 'esc' key).
  • Command mode: Used for executong commands like saving files and exiting Neovim.
  • Insert mode: Used for typing text in.
  • Visual mode: Used for highlighting text.

There are a lot of keybindings that you can use to navigate around Neovim:

In normal mode:

Keybindings Functionality
h Move left
j Move down
k Move up
l Move right
o Go to the line below
O(shift + o) Go to the line above
i Go into insert mode
a Go into insert mode, but skip one letter in a word
shift + i Go to the beginning of a line
shift + a Go to the end of a line
gg Go to the top of the content
G Go to the bottom of the content

In command mode:

Keybindings Functionality
:w Save file
:%s/word1/word2/g Substitute word1 for word2
:q Quit(exit Neovim)
:qa Exit Neovim(including all files)
:colorscheme cs1 Set a colorscheme cs1
:new Create and open a new file
:smile Try it yourself

Is Neovim free?

Neovim is a free and open-source software that comes at no cost, making it accessible to everyone. As an advanced text editor designed for developers, it builds upon the legacy of Vim with modern enhancements and an active community. Even though it is open source, donations are incredibly valuable. Contributions help sustain the project's growth, support ongoing development, and ensure that the community has access to cutting-edge tools and features.

How do I download a Neovim distribution?

There are millions of Neovim distributions available, each tailored to different workflows, programming languages, and user preferences. These distributions range from lightweight setups to highly customized configurations packed with plugins and features. If you like a certain configuration and want to download it, you can find practically any setup on GitHub or other repositories.

How does Neovim compare to VSCode?

Neovim is open-source, unlike VSCode. VSCode has a lot of Microsoft telemetry built into it, while Neovim is completely FOSS and telemetry-free. VSCode has a lot of extensions that you can download to enhance your workflow, but Neovim provides a lot more customizability, allowing users to use various package managers to download any plugin that they want. While VSCode's main focus is to be an easy-to-use editor, Neovim focuses on boosting productivity, enhanching customizability and simplicity.
